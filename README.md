# Health Score — Care Plus
### Front-End Design | Sprint 3

Interface web gamificada para incentivar a saúde preventiva dos beneficiários Care Plus.

- Eduardo Navarro Lúcio — RM: 568095

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Grid Layout e Flexbox
- Bootstrap 5.3
- Tailwind CSS v3 (CDN)
- JavaScript puro (ES6+)
- localStorage para persistência de dados

## Como rodar

Não precisa instalar nada. Basta abrir o arquivo `index.html` no navegador.

Ou acesse o deploy online: **https://sprint3-front-virid.vercel.app**

## Funcionalidades

- **Login e Cadastro** — com escolha do nome e cor do pet
- **Dashboard** — desafios diários, barra de nível, sequência e conquistas
- **Ranking** — pódio, filtro por nível (dropdown) e botão de batalha
- **Batalha de Pets** — sistema de combate elemental (Fogo, Água, Terra, Vento)
- **Clube de Vantagens** — resgate de vouchers com modal de confirmação
- **Pet Virtual** — stats de fome, higiene, alegria e energia com decay por tempo
- **Poderes Elementais** — desbloqueio por conquistas de saúde
- **Perfil** — histórico, estatísticas e logout
